# CodeAlpha_ImageGallery
Responsive Image Gallery built using HTML, CSS, and JavaScript with Lightbox, Navigation Controls, Hover Effects, and Category Filters.

A responsive image gallery built using HTML, CSS, and JavaScript.
## Features

- Responsive Grid Layout
- Lightbox View
- Next/Previous Navigation
- Hover Effects
- Smooth Transitions
- Category Filters

## Technologies Used

- HTML5
- CSS3
- JavaScript
